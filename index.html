Luigi Travi Raya

Desenvolvedor Web | Node | Git | MySQL | Typescript

Ingressei no curso técnico integrado de informática para a web no IFSul( Instituto Federal Sul-Riograndense) em 2019. Até então, adquiri o conhecimento básico em algumas linguagens e frameworks de desenvolvimento como: Java, PHP, Laravel, HTML, CSS, React, React Native, SASS, C e SQL. Além disso, aprofundei meu aprendizado, por interesse próprio, em NodeJS, Javascript e outros frameworks voltados principalmente ao desenvolvimento backend. Ingressei em 2021 no TJRS (Tribunal de Justiça do Estado do Rio Grande do Sul) como estagiário, onde realizava a manutenção de computadores e outros dispositivos eletrônicos. No inicio de 2023, fui chamado para servir no Exército Brasileiro, onde tive diversos aprendizados sobre respeito, disciplina , trabalho em conjunto e comunicação de maneira efetiva com uma equipe. Tenho a mente aberta para novas experiências e novos aprendizados, sempre visando melhorar os conhecimentos que já possuo.
